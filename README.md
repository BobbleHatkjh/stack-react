
# Stack In React

This component will realize the function like 'Stack' in flutter, You can easy to use it in react project


> 需要 node 10.13.0+  （推荐14+） 
>
> 需要 npm 6.14+

## `安装`

```
npm i stack-react
```


---

<br>



### `使用`

```
<Stack>
    <div>zIndex 1</div>
    <div>zIndex 2</div>
    <div>zIndex 3</div>
</Stack>
```
